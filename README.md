# 클로비스 수학 교구 (clovis-math)

전자칠판·블로그 수업용 인터랙티브 수학 작도 교구 모음. 전부 단일 HTML + Canvas 2D 바닐라 JS — 라이브러리·빌드 없음, 오프라인 동작.

## 공개 작품

| 작품 | 경로 | 내용 |
|---|---|---|
| 삼각형의 오심 마스터피스 | `five-centers/` | 무게중심·내심·외심·수심·방심 5탭. 컴퍼스 정통 작도(완전한 호·같은 각 표시), 꼭짓점 드래그, 자동/수동 재생, 실시간 기하 검증 카드, 방심·수심 자동 줌아웃 |
